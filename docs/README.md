## How to Edit Pages

1. Open any `.html` file in a text editor (Notepad, VS Code, TextEdit, etc.)
2. Look for text between `>` and `<` symbols — that's what shows on screen
3. Change the text and save the file
4. Upload the changed file to your hosting provider

### Common tasks

**Update a leader's name or email:**
Open `leadership.html`, find the name, change it, save.

**Add a calendar event:**
Open `calendar.html`, find the right month's `<tbody>` section,
copy an existing `<tr>...</tr>` row and paste + edit it.

**Add a photo:**
1. Copy your photo into `images/gallery/`
2. Open `gallery.html`
3. Find a placeholder `<div class="gallery-item">` block
4. Replace the `<div class="photo-placeholder">` with:
   `<img src="images/gallery/your-photo.jpg" alt="Description">`
5. Update the `data-caption="..."` attribute with your caption
6. Add more gallery items by copying an existing block

**Add a downloadable form:**
1. Copy the PDF into the `resources/` folder
2. Open `resources.html`
3. Find the right `<li>` row and update the `href` in the download link

## Making the Contact Form Work

The contact form shows a "thank you" message by default (demo only).
To make it actually send emails, choose one of:

**Option A — Formspree (free, recommended):**
1. Go to https://formspree.io → sign up → create a new form
2. Copy your Form ID (looks like `xabcdefg`)
3. In `contact.html`, change:
   `action="#"` → `action="https://formspree.io/f/xabcdefg"`
4. Remove `onsubmit="handleSubmit(event)"` from the form tag

**Option B — Netlify Forms (free if hosting on Netlify):**
Add `netlify` to the `<form>` tag. Netlify handles the rest automatically.

**Option C — mailto: (no account needed, opens email app):**
Change: `action="#" method="post"` →
`action="mailto:troop537@example.com" method="post" enctype="text/plain"`

## Customizing Colors

All colors are defined at the top of `css/style.css` as CSS variables:

```css
:root {
  --scout-green:     #2D5016;   /* main green */
  --scout-tan:       #C8A96E;   /* gold/tan accent */
  --scout-red:       #8B1A1A;   /* dark red */
  ...
}
```

Change these values to change the entire site's color scheme at once.
